# Node.js RESTful API Project

## Overview
This project implements a RESTful API using Node.js, Express, and MySQL to perform CRUD operations on a `posts` table. JSON responses are returned for all endpoints.

## Features
- GET /api/posts → Get all posts
- GET /api/posts/:id → Get one post
- POST /api/posts → Create a new post
- PUT /api/posts/:id → Update a post
- DELETE /api/posts/:id → Delete a post
- Prepared statements used for all database operations
